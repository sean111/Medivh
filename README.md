# Medivh
Helper addon to make casting portal and teleport spells easier

## Features:
* Open the search by using the command, /mv, /pm or /port
* Pre-populate the search with command arguments
* Cycle through the search results with the arrow keys

## Based on:
* PortalMaster
* PortalMasterTWW
